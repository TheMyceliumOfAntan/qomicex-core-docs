数据模块 (DataModules)

定义 Qomicex.Core 使用的核心数据结构和模型。

概述

数据模块是 Qomicex.Core 的基础，它定义了所有业务对象的数据结构。理解这些模型对于使用 Qomicex.Core 至关重要。

DataModules
├── DataDetails     # 各种详情数据类
│   ├── Account     # 账户信息
│   ├── Java        # Java 信息
│   ├── Launcher    # 启动器配置
│   └── Version     # 版本信息
├── State           # 状态码和枚举
└── SystemInfo      # 系统信息

命名空间

using Qomicex.Core;

DataDetails 命名空间

Account 类

表示 Minecraft 账户信息。

属性

属性名 类型 默认值 说明
Name string string.Empty 玩家名称
Uuid string string.Empty 玩家的 UUID（唯一标识符）
Token string string.Empty 访问令牌（用于登录验证）
AccessToken string "0" 访问令牌（备用字段）
RefreshToken string string.Empty 刷新令牌（用于续期访问令牌）
LoginMethod string "Legacy" 登录方式（Legacy、Microsoft、Yggdrasil 等）

示例

// 创建一个离线账户
var offlineAccount = new DataModules.DataDetails.Account
{
    Name = "Steve",
    Uuid = Guid.NewGuid().ToString("N"),
    Token = "",
    LoginMethod = "Offline"
};

Console.WriteLine($"创建账户: {offlineAccount.Name}");
Console.WriteLine($"UUID: {offlineAccount.Uuid}");

Java 类

表示 Java 运行时信息。

属性

属性名 类型 默认值 说明
Name string string.Empty Java 名称
Path string string.Empty Java 可执行文件的完整路径
Version string string.Empty Java 版本号（如 "17.0.2"）
VersionID int 0 版本数字标识
Type string string.Empty Java 类型（JDK、JRE 等）
Arch string string.Empty 架构（x64, x86, arm64 等）

示例

// 创建 Java 信息对象
var javaInfo = new DataModules.DataDetails.Java
{
    Name = "Java 17",
    Path = @"C:\Program Files\Java\jdk-17\bin\java.exe",
    Version = "17.0.2",
    VersionID = 17,
    Type = "JDK",
    Arch = "x64"
};

Console.WriteLine($"Java: {javaInfo.Name}");
Console.WriteLine($"路径: {javaInfo.Path}");
Console.WriteLine($"版本: {javaInfo.Version}");

Launcher 类

表示启动器配置。

属性

属性名 类型 默认值 说明
Account DataDetails.Account new DataDetails.Account() 账户信息
Java DataDetails.Java new DataDetails.Java() Java 信息
WindowSize Launcher.WindowSize new Launcher.WindowSize() 游戏窗口尺寸配置
DevideVersion bool false 是否分离版本
AdditionalParam string string.Empty 额外参数
Version string string.Empty 版本号
MaxMemory string string.Empty 最大内存分配

嵌套类 WindowSize

属性名 类型 默认值 说明
Width string "854" 窗口宽度
Height string "480" 窗口高度

示例

// 创建启动器配置
var launcherConfig = new DataModules.DataDetails.Launcher
{
    // 窗口设置
    WindowSize = new DataModules.DataDetails.Launcher.WindowSize
    {
        Width = "1920",
        Height = "1080"
    },

    // 内存设置
    MaxMemory = "4096",

    // 版本设置
    Version = "1.20.1",
    DevideVersion = false,

    // 账户信息
    Account = new DataModules.DataDetails.Account
    {
        Name = "Steve",
        Uuid = Guid.NewGuid().ToString("N"),
        LoginMethod = "Offline"
    },

    // Java 信息
    Java = new DataModules.DataDetails.Java
    {
        Path = @"C:\Program Files\Java\jdk-17\bin\java.exe",
        Version = "17.0.2"
    },

    // 额外参数
    AdditionalParam = "-Djava.net.preferIPv4Stack=true"
};

// 序列化为 JSON
var json = Newtonsoft.Json.JsonConvert.SerializeObject(launcherConfig, Newtonsoft.Json.Formatting.Indented);
Console.WriteLine(json);

Version 类

表示 Minecraft 版本信息。

属性

属性名 类型 说明
Name string 版本名称（如 "1.20.1-forge-47.0.0"）
Type string[] 版本类型数组（如 ["release", "forge"]）
GameVersion string 基础游戏版本（如 "1.20.1"）
VersionDir string 版本目录路径
State string 版本状态（如 "Available"）
StateDescribe string 状态描述（如 "Everything is OK"）

构造函数

public Version(string gameDir = "")

示例

// 创建版本信息对象
var version = new DataModules.DataDetails.Version(@"C:")
{
    Name = "1.20.1-forge-47.0.0",
    Type = new[] { "release", "forge" },
    GameVersion = "1.20.1",
    VersionDir = @"C:\Minecraft\.minecraft\versions\1.20.1-forge-47.0.0",
    State = "Available",
    StateDescribe = "Everything is OK"
};

Console.WriteLine($"版本: {version.Name}");
Console.WriteLine($"类型: {string.Join(", ", version.Type)}");
Console.WriteLine($"游戏版本: {version.GameVersion}");
Console.WriteLine($"状态: {version.State}");

State 结构体

表示状态信息。

属性

属性名 类型 默认值 说明
Name string "Available" 状态名称
Describe string "Everything is OK" 状态描述
Code int 0 状态代码

StateCode 枚举

值 说明
Available 可用（0）
MissingFiles 缺失文件（1）
WrongId 错误 ID（2）
Unknown 未知（3）

示例

// 创建状态对象
var state = new DataModules.State
{
    Name = "Available",
    Describe = "Everything is OK",
    Code = (int)DataModules.StateCode.Available
};

Console.WriteLine($"状态: {state.Name}");
Console.WriteLine($"描述: {state.Describe}");
Console.WriteLine($"代码: {state.Code}");

// 检查状态
if (state.Code == (int)DataModules.StateCode.Available)
{
    Console.WriteLine("✓ 一切正常");
}

SystemInfo 结构体

表示系统信息。

属性

属性名 类型 默认值 说明
OS string string.Empty 操作系统
OSVersion string string.Empty 操作系统版本
OSName string string.Empty 操作系统名称
Architecture string string.Empty 架构
OSVersionID string string.Empty 操作系统版本 ID

示例

// 获取系统信息
var sysInfo = SystemInfoHelper.GetSystemInfo();

Console.WriteLine("系统信息:");
Console.WriteLine($"  操作系统: {sysInfo.OS}");
Console.WriteLine($"  OS 版本: {sysInfo.OSVersion}");
Console.WriteLine($"  OS 名称: {sysInfo.OSName}");
Console.WriteLine($"  架构: {sysInfo.Architecture}");
Console.WriteLine($"  OS 版本ID: {sysInfo.OSVersionID}");

总结

数据模块是 Qomicex.Core 的基石，它定义了：

  1. Account - 账户信息的统一表示
  2. Java - Java 运行时的详细信息
  3. Launcher - 启动器配置的完整定义
  4. Version - Minecraft 版本的元数据
  5. State - 操作结果的状态信息
  6. SystemInfo - 系统硬件和软件信息

这些模型类都采用简单的属性设计，支持 JSON 序列化，便于存储和传输。
