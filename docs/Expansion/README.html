扩展模块 (Expansion)

提供与 CurseForge 和 Modrinth 两大 Minecraft 资源平台的集成支持。

概述

扩展模块为 Qomicex.Core 提供了与主流 Minecraft 资源平台的 API 对接能力，支持搜索、浏览和下载各种资源类型（Mod、整合包、资源包等）。

Expansion
├── CurseForge          # CurseForge 平台集成
│   ├── CurseForgeBase  # 基础类
│   ├── Mods            # Mod 资源
│   ├── Modpacks        # 整合包
│   ├── ResourcePacks   # 资源包
│   └── ...           # 其他资源类型
└── Modrinth            # Modrinth 平台集成
    ├── ModrinthBase    # 基础类
    ├── Mods            # Mod 资源
    ├── ModPacks        # 整合包
    └── ...           # 其他资源类型

命名空间

using Qomicex.Core.Modules.Helpers.Resources.Expansion.CurseForge;
using Qomicex.Core.Modules.Helpers.Resources.Expansion.Modrinth;

CurseForge 模块

CurseForge 是全球最大的 Minecraft 资源平台之一，提供海量的 Mod、整合包、资源包等资源。

CurseForgeBase 类

CurseForge 功能的基础类，提供 API 调用和数据获取的核心功能。

构造函数

public CurseForgeBase(string apiKey, string searchUrl, string modUrl)

使用示例

CurseForge 搜索示例

using Qomicex.Core.Modules.Helpers.Resources.Expansion.CurseForge;

// 创建 CurseForge Mods 实例
var mods = new Mods(
    apiKey: "your-api-key",
    searchUrl: "https://api.curseforge.com/v1/mods/search",
    modUrl: "https://api.curseforge.com/v1/mods"
);

// 搜索 Mod
var results = await mods.SearchAsync(
    searchFilter: "jei",
    gameVersions: new[] { "1.20.1" },
    categories: null,
    modLoaderTypes: new[] { "Forge" },
    sortField: SortField.Popularity,
    page: 1,
    pageSize: 20
);

Console.WriteLine($"找到 {results.Count} 个结果");
foreach (var mod in results)
{
    Console.WriteLine($"- {mod.Name} ({mod.DownloadCount} 下载)");
}

Modrinth 搜索示例

using Qomicex.Core.Modules.Helpers.Resources.Expansion.Modrinth;

// 创建 Modrinth Mods 实例
var mods = new Mods();

// 搜索 Mod
var results = await mods.SearchAsync(
    query: "sodium",
    projectType: "mod",
    gameVersion: "1.20.1",
    categories: null,
    loaders: new[] { "fabric" },
    index: Index.Downloads,
    page: 0,
    pageSize: 20
);

Console.WriteLine($"找到 {results.Hits} 个结果");
foreach (var mod in results.Results)
{
    Console.WriteLine($"- {mod.Title} ({mod.DownloadCount} 下载)");
}

// 获取项目详情
var projectInfo = await mods.GetProjectAsync("sodium");
Console.WriteLine($"项目: {projectInfo.Title}");
Console.WriteLine($"描述: {projectInfo.Description}");
Console.WriteLine($"作者: {projectInfo.Author}");

总结

Expansion 模块提供了完整的第三方资源平台集成能力：

  1. CurseForge - 支持全类型资源搜索和获取
  2. Modrinth - 现代化的资源平台，API 设计简洁

两个平台都支持：

  • 多类型资源（Mod、整合包、资源包等）
  • 高级搜索（按版本、加载器、分类过滤）
  • 排序和分页
  • 详细信息获取
