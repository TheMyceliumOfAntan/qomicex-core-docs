安装器模块 (Installers)

用于安装各种 Minecraft Mod 加载器的模块。

概述

安装器模块是 Qomicex.Core 的核心功能之一，它提供了统一的接口来安装各种 Minecraft Mod 加载器。无论你是需要安装 Forge、Fabric 还是其他加载器，都可以使用相同的模式来完成。

支持的 Mod 加载器

加载器 说明 适用场景
Forge 最成熟、Mod 最多的加载器 大型 Mod 包、复杂 Mod 环境
Fabric 轻量级、更新快的加载器 新版本、轻量 Mod 环境
NeoForge Forge 的继任者 未来 Forge 的替代品
Quilt Fabric 的替代品 追求更轻量的 Mod 环境
LiteLoader 轻量级客户端 Mod 加载器 客户端辅助 Mod
OptiFine 高清修复和优化 光影、性能优化

架构设计

统一接口设计

所有安装器都实现了统一的 IInstaller 接口：

public interface IInstaller
{
    /// <summary>
    /// 异步安装 Mod 加载器
    /// </summary>
    /// <param name="versionId">生成的版本ID（如 "1.20.1-forge-47.0.0"）</param>
    /// <param name="inheritsFromJson">继承自版本的JSON内容（可为空）</param>
    /// <param name="para1">参数1（根据安装器类型不同）</param>
    /// <param name="para2">参数2（根据安装器类型不同）</param>
    /// <param name="para3">参数3（可选）</param>
    /// <param name="para4">参数4（可选）</param>
    void InstallAsync(
        string versionId,
        string inheritsFromJson,
        string? para1,
        string? para2,
        string? para3,
        string? para4
    );
}

使用示例

基本安装示例

using Qomicex.Core.Modules.Helpers.Installers;

// 创建 Forge 安装器
var forgeInstaller = new ForgeInstaller(
    downloadSource: 1,           // 1 = BMCLAPI
    gameDir: @"C:\Minecraft\.minecraft",
    gameVersion: "1.20.1"
);

// 执行安装
forgeInstaller.InstallAsync(
    versionId: "1.20.1-forge-47.0.0",
    inheritsFromJson: "{ ... }",  // 原版版本 JSON 内容
    para1: @"C:\Program Files\Java\jdk-17\bin\java.exe",  // Java 路径
    para2: @"C:\Downloads\forge-1.20.1-47.0.0-installer.jar", // 安装器路径
    para3: null,
    para4: null
);

Fabric 安装示例

using Qomicex.Core.Modules.Helpers.Installers;

// 创建 Fabric 安装器
var fabricInstaller = new FabricInstaller(
    downloadSource: 1,           // 1 = BMCLAPI
    gameDir: @"C:\Minecraft\.minecraft"
);

// 异步安装 Fabric
await fabricInstaller.InstallFabricAsync(
    versionId: "1.20.1-fabric-0.14.22",
    fabricVersion: "0.14.22",
    gameVersion: "1.20.1",
    inheritsFromJson: null  // 可选
);

带进度和取消的安装（LiteLoader）

using System;
using System.Threading;
using System.Threading.Tasks;
using Qomicex.Core.Modules.Helpers.Installers;

// 创建取消令牌源
using var cts = new CancellationTokenSource();

// 创建进度报告器
var progress = new Progress<LiteloaderInstaller.InstallProgress>(p =>
{
    if (p.IsCompleted)
    {
        Console.WriteLine($"✓ {p.Message}");
    }
    else if (p.IsError)
    {
        Console.WriteLine($"✗ 错误: {p.Message}");
    }
    else
    {
        Console.Write($"\r{p.Message} - {p.Percentage}% ");
    }
});

// 创建 LiteLoader 安装器
var liteLoaderInstaller = new LiteloaderInstaller(
    sourceId: 1,
    gameDir: @"C:\Minecraft\.minecraft",
    gameVersion: "1.12.2"
);

// 执行安装
try
{
    bool success = await liteLoaderInstaller.InstallLiteLoaderAsync(
        versionId: "1.12.2-LiteLoader-1.12.2-SNAPSHOT",
        mcVersion: "1.12.2",
        liteVersion: "1.12.2-SNAPSHOT",
        progress: progress,
        cancellationToken: cts.Token
    );

    if (success)
    {
        Console.WriteLine("\n✓ LiteLoader 安装完成！");
    }
}
catch (OperationCanceledException)
{
    Console.WriteLine("\n✗ 安装已取消");
}
catch (Exception ex)
{
    Console.WriteLine($"\n✗ 安装失败: {ex.Message}");
}

总结

安装器模块提供了：

  1. 统一接口 - 所有安装器都实现 IInstaller 接口
  2. 多种加载器支持 - 支持 Forge、Fabric、NeoForge、Quilt、LiteLoader、OptiFine
  3. 灵活配置 - 支持官方源和镜像源切换
  4. 异步操作 - 支持异步安装和取消操作
