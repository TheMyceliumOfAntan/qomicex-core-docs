资源管理模块 (Resources)

提供 Minecraft 游戏资源的获取、管理和下载功能。

概述

资源管理模块负责处理 Minecraft 相关的所有资源操作，包括：

  • 获取 Minecraft 版本列表和版本清单
  • 管理本地游戏资源（库文件、资源文件、主 Jar 等）
  • 获取 Mod 加载器（Forge、Fabric 等）的版本信息
  • 检查和补全缺失的游戏文件
Resources
├── GameResourceHelper      # 游戏资源获取（版本列表等）
├── LocalResourceHelper     # 本地资源管理（库文件检查等）
└── ModLoaderResourceHelper # Mod 加载器资源管理

GetMinecraftListAsync

异步获取 Minecraft 版本列表。

public async Task<List<VersionInfo>> GetMinecraftListAsync(int source = 1)

参数： | 参数名 | 类型 | 默认值 | 说明 | |--------|------|--------|------| | source | int | 1 | 下载源（1=官方，2=BMCLAPI） |

返回值： List<VersionInfo> - 版本信息列表

示例：

var helper = new GameResourceHelper();
var versions = await helper.GetMinecraftListAsync(1);

foreach (var version in versions.Take(10))
{
    Console.WriteLine($"版本: {version.Id}, 类型: {version.Type}, 发布时间: {version.ReleaseTime}");
}

方法

GetAvailableModLoaders

异步获取可用的 Mod 加载器版本列表。

public async Task<List<ModLoaderInfo>> GetAvailableModLoaders(
    string minecraftVersion,
    ModLoaderType modLoaderType = ModLoaderType.All
)

参数： | 参数名 | 类型 | 说明 | |--------|------|------| | minecraftVersion | string | Minecraft 版本（如 "1.20.1"） | | modLoaderType | ModLoaderType | Mod 加载器类型 |

返回值： List<ModLoaderInfo> - Mod 加载器版本列表

示例：

var helper = new ModLoaderResourceHelper(1); // 使用 BMCLAPI

// 获取 Forge 版本
var forgeVersions = await helper.GetAvailableModLoaders("1.20.1", ModLoaderType.Forge);
Console.WriteLine($"找到 {forgeVersions.Count} 个 Forge 版本");

// 获取推荐版本
var recommended = forgeVersions.FirstOrDefault(v => v.IsRecommended);
if (recommended != null)
{
    Console.WriteLine($"推荐版本: {recommended.Version}");
    Console.WriteLine($"下载地址: {recommended.DownloadUrl}");
}

// 获取所有类型的 Mod 加载器
var allLoaders = await helper.GetAvailableModLoaders("1.20.1", ModLoaderType.All);
foreach (var loader in allLoaders)
{
    Console.WriteLine($"{loader.Type}: {loader.Version}");
}

总结

Resources 模块提供了完整的 Minecraft 资源管理功能：

  1. GameResourceHelper - 获取官方 Minecraft 版本列表和版本清单
  2. LocalResourceHelper - 管理本地资源，检查缺失的库文件、资源文件和主 Jar
  3. ModLoaderResourceHelper - 获取各种 Mod 加载器（Forge、Fabric、NeoForge 等）的版本信息

这些类协同工作，为 Minecraft 启动器提供完整的资源获取和管理能力。
