Qomicex.Core 启动器模块

概述

Qomicex.Core 提供了完整的 Minecraft 游戏启动功能，包括启动参数配置、游戏进程管理、日志输出等。启动器模块是 Qomicex.Core 的核心功能之一，允许开发者以编程方式启动 Minecraft 游戏。

核心组件

1. Launcher 类

Launcher 类是启动器的核心组件，负责管理游戏启动过程。

using Qomicex.Core.Modules.Launcher;

// 创建启动器实例
var launcher = new Launcher();

2. LauncherParam 类

LauncherParam 类用于配置游戏启动参数。

using Qomicex.Core.Modules.Launcher.Launcher;

// 配置启动参数
var launchParams = new LauncherParam
{
    GameDir = @"C:\Users\YourName\.minecraft", // 游戏目录
    Version = "1.20.1", // Minecraft 版本
    JavaPath = @"C:\Program Files\Java\jre-17\bin\javaw.exe", // Java 路径
    MaxMemory = 4096, // 最大内存 (MB)
    MinMemory = 1024, // 最小内存 (MB)
    WindowWidth = 1280, // 窗口宽度
    WindowHeight = 720, // 窗口高度
    Fullscreen = false, // 是否全屏
    Username = "Player", // 玩家名称
    AuthenticationToken = "your-auth-token", // 认证令牌
    Server = null, // 服务器地址 (可选)
    ServerPort = 25565 // 服务器端口 (可选)
};

启动 Minecraft

基本启动

using Qomicex.Core.Modules.Launcher;
using Qomicex.Core.Modules.Launcher.Launcher;

var launcher = new Launcher();

var launchParams = new LauncherParam
{
    GameDir = @"C:\Users\YourName\.minecraft",
    Version = "1.20.1",
    JavaPath = @"C:\Program Files\Java\jre-17\bin\javaw.exe",
    MaxMemory = 4096,
    MinMemory = 1024,
    Username = "Player"
};

// 启动游戏
var result = await launcher.StartGameAsync(launchParams);

if (result)
{
    Console.WriteLine("游戏启动成功！");
}
else
{
    Console.WriteLine("游戏启动失败！");
}

启动事件监听

using Qomicex.Core.Modules.Launcher;
using Qomicex.Core.Modules.Launcher.Launcher;

var launcher = new Launcher();

// 监听启动事件
launcher.OnGameStarted += () => Console.WriteLine("游戏已启动");
launcher.OnGameExited += (code) => Console.WriteLine($"游戏已退出，代码: {code}");
launcher.OnGameError += (error) => Console.WriteLine($"游戏错误: {error}");

// 启动游戏
var result = await launcher.StartGameAsync(launchParams);

游戏进程管理

获取运行中的游戏进程

using Qomicex.Core.Modules.Launcher;

var launcher = new Launcher();

// 获取所有运行中的游戏进程
var processes = launcher.GetRunningGameProcesses();

foreach (var process in processes)
{
    Console.WriteLine($"PID: {process.Id}, 窗口标题: {process.MainWindowTitle}");
}

停止游戏进程

using Qomicex.Core.Modules.Launcher;

var launcher = new Launcher();

// 获取所有运行中的游戏进程
var processes = launcher.GetRunningGameProcesses();

foreach (var process in processes)
{
    // 停止游戏进程
    launcher.StopGameProcess(process);
}

常见问题

游戏无法启动

问题描述：调用 StartGameAsync 后，游戏没有启动或立即崩溃。

解决方案：

  1. 检查 Java 路径是否正确：
using Qomicex.Core.Modules.Helpers;

var javaHelper = new JavaHelper();
var javaLocations = await javaHelper.SearchForJavaAsync();

if (javaLocations.Any())
{
    var firstJava = javaLocations.First();
    launchParams.JavaPath = firstJava.Path;
}
else
{
    Console.WriteLine("未找到 Java 运行时，请确保已安装 Java。");
}
  1. 检查游戏资源是否完整：
using Qomicex.Core.Modules.Helpers.Resources;

var completer = new ResourceCompleter(@"C:\Users\YourName\.minecraft");
var validationResult = await completer.ValidateResourcesAsync("1.20.1");

if (!validationResult.IsValid)
{
    Console.WriteLine("游戏资源不完整，正在修复...");
    var repairResult = await completer.RepairResourcesAsync("1.20.1");

    if (repairResult.IsSuccess)
        Console.WriteLine("资源修复成功！");
    else
        Console.WriteLine($"资源修复失败: {repairResult.ErrorMessage}");
}

游戏启动后崩溃

问题描述：游戏启动后立即崩溃或报错。

解决方案：

  1. 检查 Minecraft 版本与 Mod 加载器的兼容性。
  2. 检查是否有不兼容的 Mod。
  3. 检查内存分配是否充足。
  4. 查看游戏日志以获取详细的错误信息。

游戏进程无法停止

问题描述：调用 StopGameProcess 后，游戏进程没有立即停止。

解决方案：

  1. 尝试强制终止进程：
using Qomicex.Core.Modules.Launcher;

var launcher = new Launcher();

// 获取所有运行中的游戏进程
var processes = launcher.GetRunningGameProcesses();

foreach (var process in processes)
{
    try
    {
        // 尝试优雅地终止进程
        launcher.StopGameProcess(process);

        // 等待进程结束
        process.WaitForExit(5000);

        // 如果进程仍在运行，强制终止
        if (!process.HasExited)
        {
            process.Kill();
            process.WaitForExit();
        }
    }
    catch (Exception ex)
    {
        Console.WriteLine($"无法停止进程 {process.Id}: {ex.Message}");
    }
}

最佳实践

异常处理

在使用启动器时，应始终添加适当的异常处理：

using Qomicex.Core.Modules.Launcher;
using Qomicex.Core.Modules.Launcher.Launcher;

try
{
    var launcher = new Launcher();
    var result = await launcher.StartGameAsync(launchParams);

    if (result)
        Console.WriteLine("游戏启动成功！");
    else
        Console.WriteLine("游戏启动失败！");
}
catch (ArgumentException ex)
{
    Console.WriteLine($"参数错误: {ex.Message}");
}
catch (FileNotFoundException ex)
{
    Console.WriteLine($"文件未找到: {ex.Message}");
}
catch (Exception ex)
{
    Console.WriteLine($"启动失败: {ex.Message}");
    Console.WriteLine(ex.StackTrace);
}

日志记录

记录详细的日志信息有助于调试问题：

using Qomicex.Core.Modules.Launcher;
using Qomicex.Core.Modules.Launcher.Launcher;

var launcher = new Launcher();

// 记录调试信息
launcher.OnGameLog += (logLevel, message) =>
{
    if (logLevel == LogLevel.Debug)
        Console.WriteLine($"[DEBUG] {message}");
};

// 记录错误信息
launcher.OnGameLog += (logLevel, message) =>
{
    if (logLevel == LogLevel.Error)
        File.AppendAllText("error.log", $"[{DateTime.Now}] {message}\n");
};

资源清理

确保在游戏退出后正确清理资源：

using Qomicex.Core.Modules.Launcher;
using Qomicex.Core.Modules.Launcher.Launcher;

var launcher = new Launcher();

launcher.OnGameExited += (code) =>
{
    Console.WriteLine($"游戏已退出，代码: {code}");

    // 清理资源
    launcher.Dispose();
};

var result = await launcher.StartGameAsync(launchParams);
